Azure AI Assistant

Overview

The Azure AI Assistant is a Ghidra plugin that uses Azure OpenAI to analyze and improve function understanding.

Features

  • Find best function names based on code analysis
  • Get detailed explanations of function behavior
  • Get suggestions for better variable names

Setup

Before using the plugin, set these environment variables:

  • AZURE_OPENAI_KEY - Your Azure OpenAI API key
  • AZURE_OPENAI_ENDPOINT - Your Azure OpenAI endpoint URL

Usage

  1. Open a program in Ghidra
  2. Go to Window > Azure AI Assistant
  3. Position your cursor within a function you want to analyze
  4. Use the buttons in the Analysis tab:
    • Find Best Function Name - Get a suggested name that better describes the function's purpose
    • Explain Function - Get a detailed analysis of what the function does
    • Rename Variables - Get suggestions for clearer variable names
  5. View results in the output area below the buttons

Analysis Tab

The Analysis tab contains three main functions:

Find Best Function Name
Analyzes the current function and suggests a more descriptive name based on its behavior and purpose.
Explain Function
Provides a detailed analysis of the function including its purpose, algorithms used, and potential security implications.
Rename Variables
Suggests better names for variables based on how they are used in the code.

Output Area

The output area at the bottom of the window shows the results of AI analysis. For the Explain Function feature, the analysis is also added as a comment to the function in the listing.
