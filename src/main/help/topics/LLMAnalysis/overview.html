LLM Analysis Plugin

Overview

The LLM Analysis Plugin integrates Large Language Models into Ghidra for enhanced code analysis. It helps reverse engineers by providing automated analysis and suggestions for function renaming.

Features

  • Function Analysis - Get detailed explanations of function behavior
  • Smart Renaming - Get meaningful suggestions for function and variable names
  • Recursive Analysis - Analyze nested function calls for deeper understanding
  • Multiple Models - Configure different models for various analysis tasks

Configuration

To use the plugin, you need to configure your LLM provider settings. For Azure OpenAI:

  1. Click the "Configure LLM" button
  2. Enter your Azure OpenAI endpoint URL
  3. Enter your API key
  4. Select preferred models for analysis and renaming tasks

Usage

Function Analysis

To analyze a function:

  1. Select a function in the listing view
  2. Click "Analyze Function" for a single function analysis
  3. Enable "Recursive Analysis" to include called functions

Renaming Suggestions

To get renaming suggestions:

  1. Select a function in the listing view
  2. Click "Rename Function" for function name suggestions only
  3. Click "Rename Function & Variables" for comprehensive renaming
  4. Enable "Recursive Renaming" to include nested functions

Model Selection

Different models are optimized for different tasks:

  • Analysis Model - Used for detailed function analysis (default: GPT-4)
  • Renaming Model - Used for suggesting names (default: GPT-3.5 Turbo)
  • Recursive Model - Used for nested function analysis (default: GPT-3.5 Turbo)

Tips

  • Use recursive analysis for complex functions with multiple nested calls
  • GPT-4 provides more detailed analysis but is slower
  • GPT-3.5 Turbo is faster and suitable for quick renaming tasks
  • Consider using different models for different phases of your analysis

Note: Analysis performance depends on the model used and the complexity of the code.

Warning: Always verify suggested names and analysis results before applying them.
